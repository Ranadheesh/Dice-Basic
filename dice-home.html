Game Instructions

Start the Game:

  • Click the Start button to begin. A random number between 1 and 6 will be generated behind the scenes.
  • Guess the Number: You have 5 seconds to guess the number.
  • Enter your guess in the input field and submit.

Scoring Rules:

  • First Try: Guess the correct number to earn 5 points.
  • Subsequent Tries: If you guess wrong: A new number will be generated from the remaining numbers.
  • Correct guesses will earn decreasing points: 4 points, 3 points, and so on.
  • Game End: The game ends when only one number is left. A Restart button will appear to reset the game.
